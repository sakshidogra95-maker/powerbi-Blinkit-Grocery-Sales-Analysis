# Blinkit Grocery Sales Analysis Dashboard

## Project Overview
Interactive Power BI dashboard analyzing Blinkit grocery sales performance across regions, products, and time periods.

## Tools Used
- Power BI Desktop
- DAX for calculated metrics
- CSV sales data

## Key Features
- Total Sales, Orders, and Average Order Value
- Regional and product-wise sales analysis
- Trend analysis with interactive filters
- Insights for low-performing regions and high-growth products
